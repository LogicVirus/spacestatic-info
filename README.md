# SpaceStatic.info

Dedicated repo for the `spacestatic.info` website.

## Purpose

This project is separate from `spacestatic.com`.

Planned role for the `.info` site:
- custom agent offers
- skills and personality intake
- structured onboarding for agent builds
- future secondary app/workflow surface

## Local development

```bash
pnpm install
pnpm dev
```

Then open `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Deploy

Intended deployment target: Vercel, as its own project connected to this repo.
